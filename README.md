Round-Robin Scheduler Simulation

Overview

This project is a simulation of the Round-Robin CPU scheduling algorithm implemented in the C programming language.
The program models how an operating system scheduler distributes CPU time among multiple processes using a fixed time quantum.

The simulation demonstrates how processes are scheduled, executed, paused, and resumed until all processes finish execution.
